## About this project

Webpage used by providers to check appointments by date.

### Tecnologies

- Typescript
- React
- Styled Components

## Requirements

- [GoBarber API](https://github.com/rnanc/gobarber-api)
- Node
- Yarn

## Installing

Run the following command:

`$ yarn`

and then:

`$ yarn start`
